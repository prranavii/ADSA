## Leetcode 80 Remove Duplicates from Sorted Array II

https://leetcode.com/submissions/detail/1791660666/

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int n=nums.length;
        if(n<=2) return n;
        int i=2;
        for(int j=2;j<n ;j++){
            if(nums[j] != nums[i-2]){
                nums[i] = nums[j];
                i++;
            }
        }
        return i;
    }
}
